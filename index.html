ü§î What is Visual Basic (VB.Net)?

Visual Basic (VB) is an event-driven programming language, developed by Alan Cooper for Microsoft. This programming language is a dialect of BASIC, with important additions. Its first version was introduced in 1991, with the intention of simplifying programming. using a development environment.

The last version was 6, released in 1998, for which Microsoft extended support until March 2008.

In 2001 Microsoft. proposed to abandon the development based on the Win32 API and move to a framework or common framework of libraries, independent of the version of the operating system .NET Framework, through from Visual Basic (VB.Net) (and other languages like C Sharp (C#) for easy code transition between them); it was the successor to Visual Basic 6.

Although Visual Basic (VB.Net) is general purpose, it also provides facilities for the database application development using Data Access Objects, Remote Data Objects or ActiveX Data Objects.

Visual Basic (VB.Net) contains an integrated development environment or IDE that integrates a text editor for editing the source code, a debugger, a compiler (and linker) and a editor for graphical interfaces or GUIs. More information can be found at Wikipedia
